# Screenshots

## Extension Popup

The popup provides quick access to the main features:
- **Open Conversation Viewer**: Opens the main UI tab
- **Parse Current Page**: Parses the current ChatGPT conversation

## Main UI Tab

The main UI tab consists of:

### Left Sidebar
- Search box for filtering messages
- List of parsed conversations
- Sorted by timestamp (most recent first)

### Main Content Area

**Toolbar:**
- Selection info (shows how many messages are selected)
- Summarize button (for selected or all messages)
- Export dropdown (JSON, Markdown, Text, SQLite)

**Tree View:**
- Hierarchical display of conversation messages
- Color-coded message roles:
  - User messages (blue)
  - Assistant messages (purple)
  - System messages (gray)
- Expandable/collapsible nodes
- Click to select messages
- Ctrl+Click (Cmd+Click on Mac) for multi-select

## Features Shown

1. **Tree Navigation**: Messages displayed in parent-child relationship
2. **Search**: Real-time highlighting of search terms
3. **Selection**: Visual feedback for selected messages
4. **Export**: Multiple format options via dropdown
5. **Responsive Layout**: Sidebar and main content area
